# Smart Student Management System (Kotlin)

A modern Android mini project built with **Kotlin + MVVM + Room + RecyclerView + ViewBinding + Material 3**.

## Module highlights
- Dashboard with quick stats and navigation.
- Add/Edit/Delete/View/Search students.
- Sorting by name/marks/roll number.
- Statistics screen.
- PDF export using `PdfDocument`.
- Day/Night theme toggle.
